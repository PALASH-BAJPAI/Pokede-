## Run Locally

Clone the project

```bash
  https://github.com/PALASH-BAJPAI/PokedeX.git
```

Go to the project directory

```bash
  cd App
```

Install the requirements

```bash
  npm install
```

Run your app locally

```bash
  npm start
```
